Error: {{error.message}}

Note: if you see this Angular is probably not to blame

What to do next?

  1. Open the chrome dev tools
  2. Inspect the network tab
  3. Check the failing request
  4. Examine the request URL - ensure it is correct
  5. Reproduce the error in Postman -if we see the same reesponse, therefore issue is not with Angular

Next, the stack trace -is where our investigation should begin!

  • The first line of code in our stack trace will tell us which line of code is responsible for the error
  • What happened on this line? Why did this error occur? Then work from there ..
